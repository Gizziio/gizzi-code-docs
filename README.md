# Gizzi Code Documentation

Documentation website for **Gizzi Code** — the workspace-aware AI assistant CLI for the Allternit platform.

## Overview

This repository contains the standalone documentation site for Gizzi Code. It is built with Vite + React + Tailwind CSS and deployed independently from the main Allternit docs.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
